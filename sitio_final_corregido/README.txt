
INSTRUCCIONES PARA COMPILAR SCSS A CSS EN VISUAL STUDIO CODE
-------------------------------------------------------------

1. Abrí la carpeta del sitio web en Visual Studio Code.

2. Instalá la extensión:
   - Buscá: "Live Sass Compiler"
   - Instalá la que aparece como desarrollada por 'Ritwick Dey'

3. Ubicá el archivo:
   - scss/estilos.scss

4. Hacé clic en el botón "Watch Sass" abajo a la derecha.

5. Se generará automáticamente el archivo:
   - css/estilos.css

6. ¡Listo! Tu sitio ya estará usando los estilos compilados desde SCSS.

-------------------------------------------------------------
Consejo: No edites directamente el .css. Siempre trabajá desde el .scss
